## Home page
- **rechercher un texte dans la base**: liste des textes avec recherche à la volée -> vers page entité
- **recherche un texte ou crée un texte via URI Perseus**: champs URI -> vers page entité
- **crée un texte qui n'est pas dans la base Perseus**: disclaimer: seulement après vérification sur le catalogue Perseus -> vers page entité
- **Liste des auteurs** -> vers page auteurs
- **Liste des mots-clés** -> vers page mots-clés
- **liste des langues et des versions** -> vers page langues
- **mes contributions**: liste de tout ce qu'a fait un usager -> vers page contributions

## Page entité
- auteur, possibilité d'ajouter un auteur
- titre, possibilité d'ajouter un titre
- uri, possibilité d'ajouter un uri
- texte perseus (non éditable)
- versions, ajouter version
- alignements, ajouter un alignement
- mots-clé, possibilité d'ajouter un alignement
- images, possibilité d'ajouter une image
- scholies possibilité d'ajouter un scholie et une traduction de scholie
- notes au texte et ajouter notes
- mes contributions: ensemble de constrbutions de l'usager sur l'entité (en évidence dans les bons chanps avec bouton "éditer": pour versions: disclaimer: ça détruit les alignements)
